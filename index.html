Retro DOS Commands

Command: dir

Description: Displays a list of files and subdirectories in a directory.

Example: dir c:\ Displays the contents of the C drive.

DIR [drive:][path][filename] [/A[[:]attributes]] [/B] [/C] [/D] [/L] [/N]

Command: Help

Description: Provides Help information for Windows commands.

Example: help dir Displays Help information for the dir command.

Command: Start

Description: Starts a separate window to run a specified program or command.

Example: start notepad Opens Notepad in a new window.

Command: RMDIR

Description: Removes (deletes) a directory.

Example: rmdir /s /q foldername Deletes the folder named 'foldername' and all its contents silently.

Command: MKDIR

Description: Creates a directory.

Example: mkdir newfolder Creates a new folder named 'newfolder' in the current directory.

Command: CD

Description: Displays the name of or changes the current directory.

Example: cd Documents Changes the current directory to 'Documents'.

Command: CLS

Description: Clears the screen.

Example: cls Clears the command prompt screen.

Command: COPY

Description: Copies one or more files to another location.

Example: copy file.txt C:\destination Copies 'file.txt' to the 'C:\destination' folder.

Command: DEL

Description: Deletes one or more files.

Example: del file.txt Deletes the file named 'file.txt'.

Command: MOVE

Description: Moves one or more files from one directory to another directory.

Example: move file.txt C:\destination Moves 'file.txt' to the 'C:\destination' folder.

Command: PING

Description: Sends ICMP Echo Request messages to a specified network host to verify its availability.

Example: ping google.com Sends ping requests to google.com to check network connectivity.

Command: IPCONFIG

Description: Displays the IP configuration for all network adapters installed on the system.

Example: ipconfig /all Displays detailed information about all network adapters.

Command: TRACERT

Description: Tracks the route that packets take across a network to a specified destination.

Example: tracert google.com Traces the route to google.com and displays network latency along the path.

Command: NETSTAT

Description: Displays active TCP connections, listening ports, and network statistics.

Example: netstat -a Displays all active TCP connections and listening ports.

Command: TASKLIST

Description: Lists all running processes on the computer.

Example: tasklist Lists all running processes with their process IDs (PID).

Command: TASKKILL

Description: Terminates one or more processes by process ID (PID) or image name.

Example: taskkill /F /IM notepad.exe Terminates Notepad process forcefully.

Command: SYSTEMINFO

Description: Displays detailed configuration information about the computer's hardware and software.

Example: systeminfo Displays system information including OS version, processor, and memory.

Command: SC

Description: Communicates with the Service Control Manager and services.

Example: sc query Queries the status of all services on the system.

Command: NET

Description: Manages network resources.

Example: net use Displays information about mapped network drives.

Command: FIND

Description: Searches for a text string in files.

Example: find "searchtext" filename.txt Searches for 'searchtext' in 'filename.txt'.

Command: XCOPY

Description: Copies files and directory trees.

Example: xcopy /s source destination Copies all files and directories from source to destination.

Command: TREE

Description: Graphically displays the directory structure of a drive or path.

Example: tree /f Displays the directory tree with filenames.

Command: CHKDSK

Description: Checks a disk and displays a status report.

Example: chkdsk C: Checks the disk C: for errors and displays the report.

Command: FORMAT

Description: Formats a disk for use with Windows.

Example: format D: Formats the D: drive.

Command: SFC

Description: Scans and repairs system files.

Example: sfc /scannow Scans and repairs system files immediately.

Command: DISKPART

Description: Manages disks, partitions, and volumes.

Example: diskpart Opens the DiskPart tool for disk management.

Command: ROBOCOPY

Description: Robust File Copy for copying files and directories.

Example: robocopy sourcedir destinationdir Copies files and directories with advanced options.

Command: ASSOC

Description: Displays or modifies file extension associations.

Example: assoc .txt Displays the file type associated with .txt extension.

Command: FC

Description: Compares two files or sets of files and displays the differences between them.

Example: fc file1.txt file2.txt Compares two text files and displays the differences.

Command: GPRESULT

Description: Displays the Resultant Set of Policy (RSoP) information for a user or computer.

Example: gpresult /r Displays RSoP summary of user and computer policies.

Command: RENAME

Description: Renames a file or directory.

Example: rename oldname.txt newname.txt Renames 'oldname.txt' to 'newname.txt'.

Command: ATTRIB

Description: Displays or changes file attributes.

Example: attrib +r file.txt Sets the read-only attribute for 'file.txt'.

Command: DISM

Description: Deployment Image Servicing and Management tool for servicing Windows images.

Example: dism /online /cleanup-image /restorehealth Repairs the Windows image.

Command: FORMAT

Description: Formats a disk for use with Windows.

Example: format D: Formats the D: drive.

Command: GPUPDATE

Description: Refreshes local and Active Directory-based Group Policy settings.

Example: gpupdate /force Forces an immediate refresh of Group Policy settings.

Command: LABEL

Description: Creates, changes, or deletes the volume label of a disk.

Example: label D: MyDrive Sets the volume label of the D: drive to 'MyDrive'.

Command: NETSH

Description: Configures network settings locally or remotely.

Example: netsh interface ipv4 show interfaces Displays all IPv4 interfaces.

Command: NET USER

Description: Adds or modifies user accounts, or displays user account information.

Example: net user username password /add Adds a new user account.

Command: POWERCFG

Description: Configures power settings on a system.

Example: powercfg /hibernate off Disables hibernation.

Command: REG

Description: Modifies the Windows registry.

Example: reg query "HKEY_CURRENT_USER\Software\Microsoft" Displays information about the specified registry key.

Command: SHUTDOWN

Description: Shuts down or restarts the computer.

Example: shutdown /s /t 0 Shuts down the computer immediately.

Command: SYSTEMINFO

Description: Displays detailed configuration information about the computer's hardware and software.

Example: systeminfo Displays system information including OS version, processor, and memory.

Command: TIME

Description: Displays or sets the system time.

Example: time Displays the current system time.

Command: DATE

Description: Displays or sets the system date.

Example: date Displays the current system date.

Command: TASKLIST

Description: Lists all running processes on the computer.

Example: tasklist Lists all running processes with their process IDs (PID).

Command: ASSOC

Description: Displays or modifies file extension associations.

Example: assoc .txt Displays the file type associated with .txt extension.

Command: FC

Description: Compares two files or sets of files and displays the differences between them.

Example: fc file1.txt file2.txt Compares two text files and displays the differences.

Command: FIND

Description: Searches for a text string in files.

Example: find "searchtext" filename.txt Searches for 'searchtext' in 'filename.txt'.

Command: MORE

Description: Displays output one screen at a time.

Example: more filename.txt Displays the contents of 'filename.txt' one screen at a time.

Command: SORT

Description: Sorts the input and displays the output.

Example: sort filename.txt Sorts the lines in 'filename.txt' and displays the result.

Description: Copies files and directory trees.

Example: xcopy /s source destination Copies all files and directories from source to destination.
